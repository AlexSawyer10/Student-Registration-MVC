# Student Registration MVC

## Overview

This is a student registration website that allows you to edit, delete, and create fields within the student registration database which consists of an enrollments, courses, and students
tables. All CRUD operations are done in this project. Asynchronous operations were implemented to ensure the application remains responsive and avoids blocking the UI during database calls.

The website consists of three projects.   
-MVC Web front end  
-API project, using ASP.NET core and Entity Framwork Core  
-Shared project, Shared models to ensure API communication  

## Things Implemented

-Scaffolding  
-MVC  
-BootStrap  
-API communication
